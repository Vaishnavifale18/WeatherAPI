# WeatherAPI 🌦️

WeatherAPI is a responsive React-based weather application developed using React.js and OpenWeatherMap API. The project allows users to search for any city and view real-time weather details such as temperature, weather conditions, and wind speed through a clean and modern user interface.

## ✨ Features

* Real-time weather data
* City-based weather search
* Temperature and wind speed display
* Responsive UI design
* Error handling for invalid cities
* Modern and user-friendly interface

## 🛠️ Technologies Used

* React.js
* JavaScript
* CSS3
* OpenWeatherMap API

## 🚀 Future Enhancements

* 5-day weather forecast
* Dark mode support
* Weather icons and animations
* Search history feature
* Location-based weather detection
